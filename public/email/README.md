# E-mail Handtekening Assets

Deze map bevat de afbeeldingen voor de Crewstars e-mail handtekening.

## Beschikbare Afbeeldingen

Alle afbeeldingen zijn beschikbaar via absolute URLs op het crewstars.nl domein:

- **Logo**: `https://www.crewstars.nl/email/logo.png`
- **LinkedIn icon**: `https://www.crewstars.nl/email/linkedin.png`
- **Instagram icon**: `https://www.crewstars.nl/email/instagram.png`

## Gebruik

1. Gebruik **ALLEEN** deze absolute URLs in e-mail handtekeningen
2. Gebruik **GEEN** relatieve paden
3. Gebruik **GEEN** JavaScript
4. Alle afbeeldingen zijn PNG formaat voor maximale compatibiliteit

## Voorbeeld HTML

Zie `email-signature.html` voor een volledige HTML template die je kunt gebruiken in je e-mail client.

## Belangrijk

- Deze URLs zijn stabiel en werken in alle e-mail clients (Gmail, Outlook, Apple Mail)
- De afbeeldingen worden direct vanaf crewstars.nl geserveerd
- Geen redirects of dynamische routes - gewoon vaste bestanden
